# MakeGraph — Fast Online Chart Creation for Everyone

## What is MakeGraph?

MakeGraph is a lightweight online chart creation tool designed for people who need charts quickly.

Instead of learning complex visualization software or configuring dozens of chart settings, users can simply import data, make quick edits, and instantly generate clear, presentation-ready charts.

The goal is simple:

**Turn data into usable charts in minutes.**

---

## Why We Built It

Most charting tools fall into two categories:

### Too Complex

Many business intelligence and analytics platforms offer powerful capabilities but require significant learning time.

### Too Design-Focused

Design tools provide extensive customization options, but users often spend more time adjusting styles than communicating insights.

For most people, neither is ideal.

When someone needs a chart for a report, presentation, article, classroom assignment, or meeting, they usually want one thing:

**Get a clear chart and move on.**

MakeGraph is built for that exact scenario.

---

## Intelligent Chart Recommendations

Different data tells different stories.

MakeGraph automatically recommends chart types based on data structure:

| Data Type                      | Recommended Chart |
| ------------------------------ | ----------------- |
| Category comparison            | Bar Chart         |
| Time series                    | Line Chart        |
| Trend over time                | Area Chart        |
| Part-to-whole                  | Pie Chart         |
| Distribution and relationships | Scatter Plot      |

Users always understand why a chart is recommended.
