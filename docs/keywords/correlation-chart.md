Correlation Chart 一般指“用来展示变量之间相关关系的图表”，不是某一种唯一固定的图名，而是一类图表的统称。

- 核心目的：看两个或多个变量是不是“同增同减”、方向是否相反、关系强弱如何
- 最常见的表现形式： Scatter Plot （散点图）
- 也常见为： Correlation Matrix （相关矩阵）或 Heatmap （热力图）
  你可以这样理解：

- 正相关：一个值越大，另一个通常也越大
- 负相关：一个值越大，另一个通常越小
- 无明显相关：两个变量看不出稳定关系
  常见场景：

- 广告花费和销售额是否相关
- 学习时间和成绩是否相关
- 气温和用电量是否相关
  最典型例子是散点图：

- 每个点代表一条数据
- 点大致往右上分布，通常说明正相关
- 点大致往右下分布，通常说明负相关
- 点分布很散，通常说明相关性弱
  补充一点：

- “相关”不等于“因果”
- 两个变量看起来相关，不一定说明一个导致另一个
