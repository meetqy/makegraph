## 什么是折线图

折线图（Line Chart）通过在坐标系中标出一系列数据点，并按照既定顺序将它们连接起来，从而展示数据在连续维度上的变化情况。最常见的 X 轴是时间，例如天、周、月、季度和年；Y 轴则表示某个数值指标。

折线图最大的价值不在于展示某一个孤立点，而在于帮助读者理解整条线所反映出来的趋势、方向、节奏和波动。

如果一个问题的重点是“随时间如何变化”，折线图通常都是优先选择。

## 折线图有什么用

折线图最核心的作用，是帮助用户快速识别趋势和变化过程，尤其适合下面这些表达目标：

- 展示指标在时间维度上的上升、下降或持平
- 识别峰值、低谷、周期性波动和异常点
- 比较上线前后、活动前后或策略调整前后的变化
- 观察增长速度是否加快、放缓或反转
- 将表格中的时间序列数据转成更容易理解的视觉表达

常见例子包括：

- 网站访问量按周变化
- 月度营收或利润趋势
- 转化率、留存率、增长率变化
- 温度、能耗、传感器读数等连续数据
- 项目进度、完成率、KPI 走势

## 什么时候适合使用折线图

当你的数据满足下面这些条件时，折线图通常非常合适：

- 数据本身存在清晰的顺序，尤其是时间顺序
- 主要任务是展示变化趋势，而不是单纯比较类别
- 读者需要快速看出峰值、低谷和方向变化
- 数据点之间的连接关系有实际意义
- 你想突出连续变化而不是单个时点的绝对差异

## 什么时候不适合使用折线图

折线图并不是所有场景下的最佳选择。下面这些情况，通常应该优先考虑其他图表：

- 想比较互相独立的类别，通常 `bar chart` 更合适
- 想表达整体中的占比关系，`pie chart` 或 `stacked bar chart` 可能更合适
- X 轴标签没有自然顺序，强行连线会产生误导
- 只关心少量孤立数值，不关心变化过程
- 类别非常离散，连接后的线条无法传达有效信息

## 快速判断方法

- 如果你的问题是“它怎么变了”，优先考虑折线图
- 如果你的问题是“谁更高谁更低”，优先考虑柱状图
- 如果你的问题是“整体由哪些部分组成”，优先考虑堆叠柱状图或饼图

## 折线图与柱状图的区别

| 维度     | 折线图             | 柱状图                 |
| -------- | ------------------ | ---------------------- |
| 核心目标 | 展示趋势与变化     | 比较类别之间的数值差异 |
| 数据类型 | 有序数据、时间序列 | 分类数据、分组数据     |
| 视觉重点 | 方向、波动、拐点   | 高低、差距、排序       |
| 适合问题 | 随时间怎么变化     | 不同类别谁更高         |

## FAQ 素材

### 1. 折线图最适合用来展示什么？

折线图最适合展示一个指标如何随着时间或其他有序维度发生变化。它特别适合说明趋势、方向、波动和拐点。

### 2. 折线图适合哪些类型的数据？

最适合有顺序的数据，例如按天、按周、按月、按季度、按年，或者按步骤、阶段排列的数据。只要数据点之间存在自然顺序，折线图通常都适合。

### 3. 折线图可以展示百分比吗？

可以。转化率、留存率、增长率、完成率等百分比指标，只要是按时间或顺序变化的，都非常适合用折线图展示。

### 4. 折线图和柱状图应该怎么选？

如果主要想看趋势变化，用折线图更合适；如果主要想比较不同类别之间的数值大小，用柱状图更合适。

### 5. 什么情况下不建议使用折线图？

当 X 轴是互相独立的类别、没有自然顺序，或者重点不是变化趋势而是构成关系时，不建议优先使用折线图。

### 6. 折线图一定要用时间做 X 轴吗？

不一定。只要数据点之间存在明确顺序，例如生产流程阶段、学习进度节点、实验步骤等，也可以使用折线图。但时间序列是最常见、最自然的场景。

### 7. 折线图能显示多个系列吗？

可以。多条线可以同时展示多个指标或多个分组随时间的变化。不过系列过多时，可读性会下降，需要控制颜色和数量。

### 8. 折线图有哪些优点？

- 趋势表达清晰
- 适合展示连续变化
- 容易识别峰值、低谷和拐点
- 适合汇报和长期指标跟踪

### 9. 折线图有哪些缺点？

- 不适合比较大量独立类别
- 多条线过多时容易拥挤
- 顺序不明确时会误导读者
- 不适合强调部分与整体关系

### 10. 在线折线图工具适合哪些人使用？

适合运营、产品、市场、销售、分析师、学生、教师和内容创作者，或任何需要把时间变化数据快速整理成可视化图表的人。

### 11. 当前这个 Line Chart 工具可以做什么？

当前版本已经支持直接编辑数据、添加数据行、实时预览图表、修改标题与系列名称、切换颜色，以及控制图例和背景网格线显示，适合快速完成基础折线图制作和展示需求。
