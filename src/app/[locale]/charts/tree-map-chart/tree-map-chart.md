# 树图

## 一句话定义

树图是一种通过矩形面积展示各分类在整体中占比关系的数据可视化方式。每个矩形代表一个分类，面积越大，说明该分类的数值越高、占整体比重越大。

## 什么是树图

树图会把一个整体拆分成多个矩形区域，每个区域的面积对应一个分类的数值大小。与柱状图通过长度表达差异不同，树图通过面积表达“谁占得更多”。

它特别适合分类较多、又想在有限区域内完成整体构成展示的场景。

## 树图有什么用

树图最核心的作用，是帮助用户在一个紧凑视图里理解整体由哪些部分组成，以及各部分的相对大小。

常见表达目标包括：

- 看预算如何分配到不同部门或项目
- 看销售额由哪些产品线贡献
- 看流量由哪些渠道构成
- 看资产、库存、成本或用户来源的结构关系
- 看头部分类与长尾分类的集中程度

## 什么时候适合使用树图

当你的重点是“整体由哪些部分组成”，并且分类数量较多，希望在有限空间内同时展示很多项时，树图通常很合适。

尤其适合下面这些条件：

- 数据天然属于同一个整体
- 你关心的是各部分相对大小
- 分类数量较多，不适合堆很多柱子
- 希望快速看出头部、腰部和长尾结构
- 你更在意整体构成，而不是精确坐标比较

## 什么时候不适合使用树图

下面这些情况，通常应该优先考虑其他图表：

- 需要精确比较接近的数值，通常柱状图更合适
- 想展示时间趋势，通常折线图更合适
- 只有少量分类，树图的优势不明显
- 需要强调顺序、排名、连续变化或精确刻度

## 快速判断方法

- 如果问题是“整体由哪些部分组成”，可优先考虑树图
- 如果问题是“谁比谁高多少”，柱状图通常更清晰
- 如果问题是“随时间怎么变化”，折线图通常更合适

## FAQ 素材

### 1. 树图最适合展示什么？

最适合展示一个整体被拆分成多个部分后的构成关系，尤其适合分类较多时做紧凑展示。

### 2. 树图和柱状图怎么选？

如果你更在意整体构成和空间利用率，树图更合适；如果你更在意精确比较和排序，柱状图通常更好读。

### 3. 树图可以展示百分比吗？

可以。只要每个值都代表整体中的一个部分，不管是原始值还是百分比，都可以用树图表达。

### 4. 树图为什么适合分类很多的场景？

因为它把很多分类压缩进同一个矩形区域中，不需要像柱状图那样为每个类别保留独立轴空间。

### 5. 树图什么时候会难读？

当分类太多且很多值过小、过接近，或者你需要精确比较时，树图会变得不够直观。

### 6. 树图里的数值需要能加总吗？

通常建议是的。树图在表达同一整体构成时最有意义。
